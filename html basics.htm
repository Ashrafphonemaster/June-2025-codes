Click CTRL + U in an HTML page, or right-click on the page and select "View Page Source".This will open a new tab containing the HTML source code of the page. Right-click on an element (or a blank area), and choose "Inspect" to see what elements are made up of (you will see both the HTML and the CSS). You can also edit the HTML or CSS on-the-fly in the Elements or Styles panel that opens. HTML images are defined with the img> tag. The source file (src), alternative text (alt), width, and height are provided as attributes. The link's destination is specified in the href attribute. HTML elements with no content are called empty elements. The br> tag defines a line break, and is an empty element without a closing tag:
